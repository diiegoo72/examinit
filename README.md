# Examinit

Examinit is a web platform for creating, managing and analyzing multiple-choice exams.

The project is an evolution of a previous Java console application developed for a university programming course. The goal is to transform it into a professional backend/web application using modern software engineering practices.

## Planned features

- Question bank management
- Exam creation
- Exam attempts
- Automatic grading
- Question statistics
- Exam analytics
- Import/export
- REST API
- Web interface

## Tech stack

- Java 21
- Spring Boot
- PostgreSQL
- Spring Data JPA
- Flyway
- JUnit
- Docker
- GitHub Actions
