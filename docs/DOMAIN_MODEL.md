# Domain Model

## Main entities

- User
- Question
- AnswerOption
- Exam
- ExamQuestion
- ExamAttempt
- StudentAnswer

## Initial notes

The first version of Examinit will focus on the core exam flow:

1. A teacher creates questions.
2. A teacher creates an exam.
3. A student starts an exam attempt.
4. The student submits answers.
5. The system calculates the score.
6. The teacher can review basic statistics.
